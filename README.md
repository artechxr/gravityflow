# GravityFlow | Smart Stadium Ecosystem

## 🏟️ Problem Statement Alignment
GravityFlow is designed to improve the **physical event experience** for attendees at **large-scale sporting venues**. It directly addresses:
* **Crowd Movement:** Dynamic heatmaps for bottleneck prevention.
* **Waiting Times:** AI-driven **Predicted Waiting Time** logic for queues.
* **Real-time Coordination:** Integrated Staff & Vendor communication grid.

## 🛠️ Tech Stack & Google Integration
* **Frontend:** Optimized HTML5, CSS3 (Glassmorphism), JavaScript.
* **Backend Logic:** Integration with **Google Apps Script** for live data logging.
* **Testing:** Modular JS testing for throughput and flow logic.

## 🚀 Key Features
1. **Smart Routing:** Real-time directional prompts.
2. **Virtual Queuing:** Digital line management to reduce physical wait.
3. **Staff Dashboard:** Live crowd analytics via simulated CCTV feeds.
